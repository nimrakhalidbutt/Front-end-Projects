## 📖 Description

**Brew & Bloom** is a **premium, visually immersive static landing page** designed for a modern coffee shop. It blends **elegant typography, subtle animations, interactive particle effects, and a dark, warm aesthetic** to create a memorable brand experience.

### ✨ Key Features

- **Particle Background System** — Interactive floating particles that respond to mouse movement
- **Premium Typography** — Clean hierarchy using `DM Serif Display` (headlines) and `Inter` (body text)
- **Responsive Grid Layout** — CSS Grid for macro structure, Flexbox for micro components
- **Image Optimization** — AVIF/WebP formats with explicit dimensions for performance
- **Smooth Animations** — Subtle fade-ups, slide-ins, and hover interactions
- **Glassmorphism Effects** — Frosted glass panels with backdrop blur
- **Semantic HTML5** — Proper use of `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- **DRY CSS** — Utility classes and reusable styles
- **W3C Valid & Lighthouse Ready** — Built with best practices
